    N4-04 单词

    单词 假名 声调 词品 词义 例句
    父 ちち 1/2 名词 父亲 父に写真を見せます。
    我向父亲展示照片。
    写真 しゃしん 0 名词 照片；摄影
    見せます みせます 3 他动词・一段/二类 给……看
    子ども こども 0 名词 儿女（自己的孩子）；小孩（年幼的孩子） 子どもにピアノを教えます。
    我教孩子弹钢琴。
    ピアノ ぴあの 0 名词 钢琴
    Piano
    教えます おしえます 0 他动词・一段/二类 教；指点
    結婚 けっこん 名・自动词・サ变/三类 结婚 私は美智子さんと結婚したいです。
    我想和美智子结婚。
    たい たい 1 助动词 想；表示希望做什么
    です です 1 助动词 是
    兄 あに 1 名词 哥哥 兄と喧嘩をしました。
    我和哥哥吵架了。
    喧嘩 けんか 0 名・自动词・サ变/三类 吵架；打架
    しました しました 2 他动词・サ变/三类 做；干
    先生 せんせい 3 名词 老师 先生に相談します。
    我去找老师商谈。
    相談します そうだんします 6 他动词・サ变/三类 商谈；商量；讨论
    会います あいます 3 自动词・五段/一类 见面 先生に会います。
    我遇见了老师。
    わたしはきょうリーさんと会います。
    我今天会与李先生见面。
    友だち ともだち 0 名词 朋友 友だちと相談します。
    我和朋友商量。
    友だちと会います。
    我和朋友见面。
    旅行 りょこう 0 名・自他・サ变/三类 旅行 林さんに旅行のことを聞きます。
    我想问问林先生旅行的事。
    のこと のこと 3 名词 的事；的内容
    聞きます ききます 3 他动词・五段/一类 听
    友達 ともだち 0 名词 朋友 友達とよく話し合って旅行の日を決めてください。
    请您与朋友好好商量后决定旅行日期。
    よく よく 1 副词 认真地
    話し合って
    原「話し合う」    		 はなしあって 4 他动词・五段/一类 商量；讨论
    日 ひ 1/0 名词 日期
    決めて
    原「決める」    		 きめて 0 他动词・一段/二类 决定；商定
    買い物 かいもの 0 自动词・サ变/三类 买东西 デパートへ買い物に行きます。
    我去百货商店买东西。
    あした あした 0 名词 明天 あした留学生がこの工場の見学に来ます。
    明天，留学生们来参观这个工厂。
    留学生 りゅうがくせい 3/4 名词 留学生
    工場 こうじょう 3 名词 工厂
    見学 けんがく 0 名・他动词・サ变/三类 参观学习
    来ます きます 2 自动词・カ变/三类 来，到来
    このへん このへん 0 名词 这地方（一小片区域） このへんは買い物に便利です。
    这地方购物很方便。
    便利 べんり 1 名・形容动词/ナ形容词 便利；方便
    ビザ びざ 1 名词 （护照）签证
    Visa    		 ビザの延長にどんな書類が必要ですか。
    我需要什么文件来延长我的签证？
    延長 えんちょう 0 名・他动词・サ变/三类 延长；扩展
    どんな どんな 1 连体词 如何
    書類 しょるい 0 名词 文件；档案；资料
    必要 ひつよう 0 名・形容动词/ナ形容词 必须；需要
    事故 じこ 0 名词 事故 事故で電車が止まっています。
    电车因事故而停止。
    止まって とまって 0 自动词・五段/一类 停止
    毎日 まいにち 1 名词 每天；天天 毎日雨でテニスの練習ができません。
    因为每天都下雨所以无法练习网球。
    雨 あめ 1 名词 雨；下雨；雨天
    テニス てにす 1 名词 网球
    Tennis
    練習 れんしゅう 0 名・他动词・サ变/三类 练习，反复学习
    でき でき 1/0 名词 完成
    バス ばす 1 名词 公交车
    Bus    		 バスで学校へ行きます。
    乘公共汽车去学校。
    ペン ぺん 1 名词 笔
    Pen    		 ペンで書いてください。
    请用笔写。
    英語 えいご 0 名词 英语 英語で手紙を書きます。
    用英文写一封信。
    手紙 てがみ 0 名词 书信
    バター ばたー 1 名词 黄油
    Butter    		 このバターと卵でおいしいケーキを焼きます。
    用这些黄油和鸡蛋烘烤美味的蛋糕。
    卵 たまご 2/0 名词 鸡蛋
    おいしい
    美味しい    		 おいしい 3/0 形容词/イ形容词 好吃；好喝
    ケーキ けーき 1 名词 蛋糕
    Cake
    焼きます やきます 3 他动词・五段/一类 烤；烧
    紙 かみ 2 名词 纸 紙で人形を作ります。
    用纸做一个娃娃。
    人形 にんぎょう 0 名词 娃娃；玩偶
    作ります つくります 4 他动词・五段/一类 制作
    日本酒 にほんしゅ 0 名词 日本酒 日本酒は米から作ります。
    日本酒是用大米酿造的。
    米 こめ 2 名词 稻米；大米
    石油 せきゆ 0 名词 石油 石油からいろいろなものができます。
    石油可以制成各种东西。
    できます できます 1 动词 能够做/可以
    はさみ はさみ 2/3 名词 剪刀 このはさみで花を切ってください。
    请用这把剪刀修剪花朵。
    切って
    原「切る」    		 きって 1 他动词・五段/一类 剪
    竹 たけ 0 名词 竹 わたしは竹で作ったいすとテーブルが好きです。
    我喜欢用竹子制作的椅子和桌子。
    作った
    原「作る」    		 つくって 2 他动词・五段/一类 制造
    いす いす 0 名词 椅子
    テーブル てーぶる 0 名词 桌子
    Table
    もっと もっと 1 副词 更加 もっと大きい声で話してください。
    请用更大声音说话。
    大きい おおきい 3 形容词/イ形容词 大
    声 こえ 1 名词 声
    怪我 けが 2 名词 受伤 私は怪我で 1 週間入院しました。
    我因为受伤住院了一个星期。
    週間 しゅうかん 0 名词 一个星期
    入院しました
    原「入院」    		 にゅういんしました 0 自动词・サ变/三类 住（医）院
    見舞い みまい 0 名词 慰问 友だちがお見舞いに来てくれました。
    朋友来探望我了。
    来て
    原「来る」    		 きて 0 动词・て形 来了
    くれました
    原「くれます」    		 くれました 0 他动词・一段/二类 给予某种行动
    来てくれました 整段可以翻译成：
    来了（为我）
    健康 けんこう 0 名・形容动词/ナ形容词 健康 何か健康にいいことをしていますか。
    你在做对你的健康有益的事情吗？
    いいこと いいこと 1 名词 好事
    しています しています 0 动词短语 正在做某事
    尾部加 か 表示：正在做某事吗？
    授業 じゅぎょう 1 名・自动词・サ变/三类 授课；上课 授業は 9 時に始まります。
    上午 9 点开始上课。
    始まります はじまります 5 自动词・五段/一类 开始
    来週 らいしゅう 0 名词 下周 来週父が日本へ来ます。
    我爸爸下周来日本。
    銀行 ぎんこう 0 名词 银行 銀行は 9 時から 3 時までです。
    银行营业时间为上午 9 点至下午 3 点。
    正式场合或者书面表达：
    銀行の営業時間は午前 9 時から午後 3 時までです。

    原「降る」    		 ふっています 0 自动词・五段/一类 （雨、雪等）下 朝から雨が降っています。
    从早上开始一直在下雨。
    ごろ ごる 1 接尾词 ……左右 きのうは 2 時ごろまでずっと本を読んでいました。
    昨天我看书到大约 2 点左右。
    ずっと ずっと 0 副词 一直
    読んでいました
    原「読む」    		 よんでいました 1 他动词・五段/一类 读
    申し込み もうしこみ 0 名词 申请 25 日までに申し込み用紙を出してください。

    用紙 ようし 0/1 名词 （特定用途的）纸张
    出して
    原「出す」    		 だして 1 他动词・五段/一类 拿出；提交；寄出
