    N4-06 单词

    单词 假名 声调 词品 词义 例句
    これ これ 0 代词 这 これはあなたの本ですか。
    这是你的书吗？
    あなた あなた 1 代词 你
    いいえ いいえ 3 感叹词 不，不是，没有，略表敬意的否定回答的用语 いいえ、これはリンさんのです。
    不，这是林先生的。
    だれ（誰） だれ 1 代词 谁 これはだれのですか。
    这是谁的？
    それ それ 0 名词 代替前面说过的；那个 それはリンさんのです。
    这是林先生的。
    この この 1 连体词 这个 この方はどなたですか。
    这是哪位？
    方 かた 2 名词 位
    どなた どなた 1 代词 谁，哪位（比较礼貌）
    あそこ あそこ 0 代词 代词 あそこでちょっと休みましょう。
    我们在那里稍微休息一下吧。
    ちょっと ちょっど 0 副词 稍微
    休みましょう
    原「休む」    		 やすみましょう 2 自他・五段/一类 休息
    ええ ええ 1/2 感叹词 嗯；啊；欸（ei） ええ、あの木の下がいいですね。
    嗯，那棵树底下挺好的。
    あの あの 0 连体词 那个
    木 き 1 名词 树；木材
    いいですね いいですね 惯用句 挺好；不错
    きのう（昨日） きのう 2 名词 昨天 きのう駅前の花屋で花を買ったんだけど……
    昨天在车站前的花店买了花，但是…
    花屋 はなや 2 名词 花店
    だけど んだけど 2 接续词 但是，可是，转折性的接续词
    ああ ああ 0 感叹词 啊 ああ、あの店ね。あそこは、店員さんが親切だね。
    啊，那家店呢。那家店，店员很亲切呢。
    店員 てんいん 0 名词 店员
    親切 しんせつ 1 名・形容动词/ナ形容词 亲切
    ね / だね 终助词 表示感叹；表示确认；表示同意
    弟 おとうと 4 名词 弟弟 弟は今パン屋でアルバイトをしています。
    我弟弟现在在面包店打工。
    今 いま 1 名词 现在
    パン ぱん 1 名词 面包
    屋 や 1 名词 房屋
    アルバイト あるばいと 3 名・自动词・サ变/三类 打工
    しています
    原「する」的正在进行时    		 しています 0 他动词・サ变/三类 干；做；办
    以上的正在进行时
    とても とても 0 副词 非常 そのパン屋のパンはとてもおいしいんです。
    那家面包店的面包非常好吃。
    おいしい おいしい 0 形容词/イ形容词 好吃；可口；美味
    んです
    原「のです」口语形式    		 んです 1 助词 （口语）是的
    娘 むすめ 3 名词 女儿；闺女 わたしの娘の名前はゆり子です。この子は今、幼稚園に通っています。
    我的女儿叫做 Yuriko。这个孩子现在正在上幼儿园。
    名前 なまえ 0 名词 名
    子 こ 0 名词 孩子
    幼稚園 ようちえん 3 名词 幼儿园
    通っています
    原「通う」    		 かよっています 0 自动词・五段/一类 来往，（在两个地点之间）往返
