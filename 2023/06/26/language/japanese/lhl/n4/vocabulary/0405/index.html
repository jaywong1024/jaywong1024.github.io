    N4-05 单词

    单词 假名 声调 词品 词义 例句
    伝えたい つたえたい 0 动词 想要传达 伝えたい情報の前。
    在要传达的信息之前。
    情報 じょうほう 0 动词 情报；信息
    前 まえ 1 名词 前
    後 あと 1 名词 后 伝えたい情報の後。
    在要传达的信息之后。
    時計 とけい 0 名词 钟，表 これは時計です。
    这是手表。
    だれ（誰） だれ 1 代词 谁 だれが来ましたか。
    谁来了？
    映画 えいが 0/1 名词 电影 3 年前にあの映画を中国で見ました。
    3 年前我在中国看过那部电影。
    見ました みまして 2 他动词・一段/二类 看，过去式
    お酒 おさけ 0 名词 清酒 お酒は好きなんですが 、すぐ顔が赤くなってしまうんです。
    虽然我喜欢喝酒，但一喝酒就会脸红。
    好きなんですが すきなんですか 虽然喜欢，但是～
    这里的「が」是一个转折连接助词，用于引出后面的对比或限制
    すぐ すぐ 1 副词 立即
    顔 かお 0 名词 脸
    赤く
    原「赤い」    		 あかく 0 形容词/イ形容词 红色的
    なってしまう
    原「なる」    		 なってしまう 1 自动词・五段/一类 动词「なる」的一种完成形式，表示动作的完成或不可避免
    うち うき 0 名词 家里 わたしのうちに犬はいません。
    我家里没有狗。
    犬 いぬ 2 名词 狗
    いません いません 0 自动词 「いる」的否定形式，表示「没有」
    結婚しません けっこんしません 0 名 不结婚 わたしはあの人とは結婚しません。
    我不会与这个人结婚的。
    あ あ 1 感叹词 啊，呀，噢 あ、 鳥が水を飲んでいます。
    呀，鸟在喝水。
    鳥 とり 0 名词 鸟
    水 みず 0 名词 水
    飲んでいます のんでいます 1 他动词・五段/一类 喝；饮；吃
    降ってきました
    原「降る」    		 降って ふってきました 自动词・五段/一类 （雨、雪等）下，落 雨が降ってきましたよ。
    下起雨来了呢。
    よ よ 语气词 用于强调或提醒对方
    外 そと 1 名词 外面 家の外は寒いですが 、中は暖かいです。
    屋外很冷，但屋内很暖和。
    寒い さむい 2 形容词/イ形容词 冷；寒碜
    中 なか 1 名词 里边
    暖かい あたたかい 4 形容词/イ形容词 暖和
    ワイン わいん 1 名词 葡萄酒
    Wine    		 ワインは飲めますが 、ウイスキーは飲めません。
    我可以喝葡萄酒，但不能喝威士忌。
    飲めます
    原「飲める」    		 のめます 2 他动词・一段/二类 能喝；好喝
    飲めません 同上，否定形式：不能喝
    ウイスキー ういすきー 3 名词 威士忌酒。洋酒的一种
    Whisky
    子 こ 1 名词 子女 この子はうちでよく話しますが、外ではあまり話しません。
    这孩子在家话很多，但在外边不怎么讲话。
    よく よく 1 副词 经常地
    話します はなします 4 話します 讲话
    話しません 同上，否定形式：不讲话
    あまり あまり 0 副词 不怎么；不太
    昼ご飯 ひるごはん 3 名词 午饭 昼ご飯（を）は 、食堂で食べます。食堂では安く食べられるからです。
    我在食堂吃午饭，因为在食堂吃午饭便宜。
    食堂 しょくどう 0 名词 食堂
    食べます
    原「食べる」    		 たべます 3 他动词・一段/二类 吃（敬体形式）
    安 やす 0 接头/接尾 便宜；轻率
    られる られる 2 接续词 表示可能：能够
    から から 1 接续词 简体形/敬体形+から
    因为～
    料理 りょうり 1 名・他动词・サ变/三类 烹调；料理 料理（が）は下手なんですが 、作ること（が）は好きです。日曜日には必ず作ります。
    虽然我不擅长做饭，但我喜欢做饭。在星期日，我一定会做饭。
    下手な へたな 2 形容词 不擅长
    作る つくる 2 他动词・五段/一类 作，造，制，将材料加工成其他东西
    こと こと 2 名词 事情
    日曜日 にちようび 3 名词 星期日
    必ず かならず 0 副词 一定
    部屋 へや 2 名词 房间 この部屋は窓が大きいですね。
    这个房间的窗很大呢。
    窓 まど 1 名词 窗户
    大きい おおきい 3 形容词/イ形容词 大
    車 くるま 0 名词 车 わたしは車がほしいです。
    我想要一辆车。
    ほしい
    欲しい    		 ほしい 2 形容词/イ形容词 想要
    頭 あたま 2 名词 头 わたしは頭が痛いです。
    我头疼。
    痛い いたい 2 形容词/イ形容词 疼的；痛苦的
    歌 うた 2 名词 歌 ゆみこさんは歌が上手です。
    由美子唱歌很好听。
    上手 じょうず 3 名・形容动词/ナ形容词 好；擅长
    力 ちから 3 名词 力量；实力 この子は力があります。
    这孩子很有实力。
    あります
    原「ある」    		 あります 3 自动词・五段/一类 有；存在
