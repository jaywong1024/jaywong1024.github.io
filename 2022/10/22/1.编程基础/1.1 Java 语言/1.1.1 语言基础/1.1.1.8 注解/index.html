    参考摘抄：《Java 编程思想》

    一、基本语法

    1.1 定义注解

    下面是一个 Java 开发中最常用到的注解之一，lombok 包的 @Data 注解的定义。它的作用是为所有字段生成 getter 和 setter 方法，以及类的 toString、hashCode 和 Constructor 方法

    可以看到定义注解类似于定义一个接口，在定义注解时需要用到一些 元注解 来说明它的 应用类型 和 保存级别

    1.2 元注解

    注解 描述
    @Target 表示注解可以用于什么地方（多个），值：java.lang.annotation.ElementType
    - TYPE：用于类、接口（包括注解类型）或枚举
    - FIELD：用于字段（包括枚举常量）
    - METHOD：用于方法
    - PARAMETER：用于正式参数
    - CONSTRUCTOR：用于构造方法
    - LOCAL_VARIABLE：用于局部变量
    - ANNOTATION_TYPE：用于枚举
    - PACKAGE：用于包装
    @Retention 表示需要在什么级别保存该注解信息（单个），值：java.lang.annotation.RetentionPolicy
    - SOURCE：源码级别，编译后丢弃
    - CLASS：释将由编译器记录在类文件中，但不必在运行时由 VM 保留（默认级别）
    - RUNTIME：注解将由编译器记录在类文件中，并在运行时由 VM 保留，因此可以反射性地读取它们
    @Documented 将此注解包含在 Javadoc 中
    @Inherited 允许子类继承父类中的注解

    1.3 注解不支持继承

    注解不能使用关键字 extends 来继承某个 @interface，一般采取嵌套的方式，例如 Spring 的 @Service 注解就包含 @Component，@RestController 包含 @Controller 和 @ResponseBody
